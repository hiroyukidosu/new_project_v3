# 10年運用評価レポート

## 📊 現状分析

### ✅ 良い点

1. **月次セグメンテーション実装済み**
   - 服用ステータス（`dose_status`、`weekday_status`、`memo_enabled`）は月単位で分割保存
   - 10年 = 120ヶ月分のデータを効率的に管理可能

2. **Hive使用**
   - 高速なNoSQLデータベース
   - 大量データにも対応可能

3. **バッチ処理実装**
   - 50件ずつ処理してフレーム分散
   - UIブロッキングを防止

4. **バックアップ機能**
   - 3重バックアップでデータ保護

### ⚠️ 問題点

1. **服用メモが全件読み込まれている**
   ```dart
   // 現状: 全件読み込み
   final allValues = box.values; // 10年分全て読み込む
   ```
   - **影響**: 10年分だと数千〜数万件になり、メモリと起動時間に影響
   - **推定データ量**: 
     - 1日10件 × 365日 × 10年 = 36,500件
     - 1件あたり約500バイト = 約18MB（メモリ）

2. **日付フィルタリングが未実装**
   ```dart
   // MedicationRepository._loadMemosInIsolate
   return true; // 一旦全て読み込む（後で最適化）← 実装されていない
   ```

3. **古いデータのアーカイブ機能がない**
   - 古いデータを自動削除する機能がない
   - データが無制限に蓄積される

4. **パフォーマンス懸念**
   - 起動時に全件読み込むため、データ量に比例して起動時間が増加
   - メモリ使用量が増加

## 💡 改善提案

### 🔴 最優先（必須）

#### 1. **服用メモの日付フィルタリング実装**

```dart
// MedicationRepository._loadMemosInIsolate を修正
static List<MedicationMemo> _loadMemosInIsolate(Box<MedicationMemo> box) {
  final cutoffDate = DateTime.now().subtract(const Duration(days: 365 * 2)); // 2年分のみ
  return box.values
      .where((memo) {
        // createdAtでフィルタリング
        return memo.createdAt.isAfter(cutoffDate);
      })
      .toList();
}
```

**効果**: 
- 起動時の読み込み件数を大幅削減（36,500件 → 7,300件）
- メモリ使用量を80%削減

#### 2. **古いデータのアーカイブ機能**

```dart
// MedicationDataPersistence に追加
Future<void> archiveOldMemos({int keepYears = 2}) async {
  final cutoffDate = DateTime.now().subtract(Duration(days: 365 * keepYears));
  final box = Hive.box<MedicationMemo>('medication_memos');
  
  final oldMemos = box.values
      .where((memo) => memo.createdAt.isBefore(cutoffDate))
      .toList();
  
  // アーカイブボックスに移動
  final archiveBox = await Hive.openBox<MedicationMemo>('medication_memos_archive');
  for (final memo in oldMemos) {
    await archiveBox.put(memo.id, memo);
    await box.delete(memo.id);
  }
}
```

**効果**: 
- アクティブなデータ量を制限
- パフォーマンスを維持

### 🟡 推奨（パフォーマンス向上）

#### 3. **ページネーション実装**

```dart
// メモ一覧をページ単位で読み込む
Future<List<MedicationMemo>> getMemosPaginated({
  int page = 0,
  int pageSize = 50,
  DateTime? startDate,
  DateTime? endDate,
}) async {
  final box = Hive.box<MedicationMemo>('medication_memos');
  final allMemos = box.values
      .where((memo) {
        if (startDate != null && memo.createdAt.isBefore(startDate)) return false;
        if (endDate != null && memo.createdAt.isAfter(endDate)) return false;
        return true;
      })
      .toList()
    ..sort((a, b) => b.createdAt.compareTo(a.createdAt));
  
  final start = page * pageSize;
  final end = (start + pageSize).clamp(0, allMemos.length);
  return allMemos.sublist(start, end);
}
```

**効果**: 
- 必要な分だけ読み込む
- メモリ使用量を最小化
